The Food Junction is a simple, elegant restaurant webpage built using **HTML**, **CSS**, and **JavaScript**. It showcases the restaurant's offerings, ambiance, and services in a visually appealing way.

- **Responsive Design**: Optimized for various screen sizes.
- **Interactive UI**: Smooth navigation and user-friendly interface.

- **Contact Section**: Includes contact details and a feedback form.

## Technologies Used

- **CSS**: For styling and layout.
- **JavaScript**: For interactivity and dynamic elements.
